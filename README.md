# HTML Final home work - Burger app creation

## 1. Project Overview

**Project name:** BurFull Burger App  
**Type:** Front-end homework project (HTML & CSS)  
**Time:** HTML/Autumn/2025  

This project is a static website created as part of an HTML and CSS final homework assignment.  
The main goal of the project was to practice semantic HTML structure, CSS styling, and basic layout techniques.

The project focuses on:
- Structuring web pages using semantic HTML
